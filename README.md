# Bezier playground  ✍️

Bezier playground implements basic bezier curve editor features, like:
- add multiple bezier curves to a spline
- add multiple splines to a canvas
- change spline colors
- change spline width
- remove spline
- [figma](https://figma.com) like drawing UX
